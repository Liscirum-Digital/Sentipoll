Herzlich willkommen bei Tubaerit!

Tubaerit ist ein Open-Source-Tool, mit dem Daten von lokalen Gruppen gesammelt werden können.
Nach der Erstellung einer Umfrage können die Benutzer auf eine einfache Weboberfläche zugreifen und Messdaten bereitstellen. Die gesammelten Informationen werden in einem Diagramm dargestellt, können aber auch in eine CSV-Datei exportiert werden.
Das Projekt wird von Liscirum Digital für CG entwickelt.
